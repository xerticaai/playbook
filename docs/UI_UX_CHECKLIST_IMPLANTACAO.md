# Checklist de Implantação UI/UX (Pré-código)

## Gate 0 — Planejamento
- [ ] Documento mestre aprovado
- [ ] Sprint 00 aprovada
- [ ] Escopo Sprints 1..5 congelado

## Gate 1 — Preparação técnica
- [ ] Mapeamento de componentes afetados
- [ ] Mapeamento de contratos de dados (Net, Meta/Budget, SLA)
- [ ] Plano de rollback visual definido

## Gate 2 — Execução por sprint
- [ ] Sprint 1 entregue e validada
- [ ] Sprint 2 entregue e validada
- [ ] Sprint 3 entregue e validada
- [ ] Sprint 4 entregue e validada
- [ ] Sprint 5 entregue e validada

## Gate 3 — Validação final
- [ ] UX review com stakeholders
- [ ] Coerência de filtros e drill-down
- [ ] Verificação responsiva e acessibilidade mínima
- [ ] Publicação com changelog
